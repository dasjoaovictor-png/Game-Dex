# 🎮 Game-Dex
Game-Dex is a **school project** built using simple front-end technologies, along with a few imported fonts and libraries.

| HTML | CSS | JS | NODE | REACT |
| :---: | :---: | :---: | :---: | :---: |
| :heavy_check_mark: | :heavy_check_mark: | :heavy_check_mark: | :heavy_multiplication_x: | :heavy_multiplication_x: |

Designed to provide a simple and user-friendly virtual game store, the project includes the core features of an e-commerce platform:

- **Create an account** (includes an administrator account)
- **Add games to the catalog** (administrator-only feature)
- **Search for specific games** using the search bar
- **Add games to favorites**
- **Add games to the shopping cart** with automatic price calculation
- **Complete a purchase**
